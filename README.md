# 🚀 HACK THE FUTURE - Sitio Web Oficial

## 📋 Descripción del Proyecto

Hack The Future es una iniciativa social que busca capacitar a **120 jóvenes de Manrique** en habilidades digitales, inteligencia artificial y emprendimiento para acceder a la economía global.

Este repositorio contiene el sitio web oficial completo con 10 páginas profesionales, optimizado para performance, accesibilidad y SEO.

## 🎯 Objetivos del Sitio Web

- ✅ **Presencia Digital Profesional**: Establecer una imagen sólida y confiable
- ✅ **Participación Comunitaria**: Facilitar la participación de jóvenes, mentores y empresas
- ✅ **Transparencia**: Mostrar impacto social y resultados del proyecto
- ✅ **Confianza**: Generar confianza con stakeholders e inversores
- ✅ **Accesibilidad**: Garantizar acceso universal a la información

## 🚀 Inicio Rápido

### **Prerrequisitos**
- Node.js (versión 14 o superior)
- npm o yarn

### **Instalación**
```bash
# Clonar repositorio
git clone [repository-url]
cd Social-Project-Hack-The-Future

# Instalar dependencias
npm install

# Construir CSS
npm run build:css

# Servir localmente
npm run serve
```

### **Scripts Disponibles**
```bash
npm run dev          # Modo desarrollo con watch
npm run build        # Build completo
npm run serve        # Servir en puerto 3000
npm run start        # Build + serve
npm run test:lighthouse  # Testing de performance
```
